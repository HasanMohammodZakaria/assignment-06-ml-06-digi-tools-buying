###### **Name of your  Project**

--- Digital Tools Buying Website

###### **Project Description**

--- Digital Tools Buying is a modern web application where users can explore and purchase useful digital tools easily. The goal of this project is to provide a clean, fast, and user-friendly experience for discovering tools that can improve productivity and workflow.

This project focuses on simplicity, smooth UI, and practical functionality — making it easy for users to browse products, add them to cart, and view their selections in real-time.

###### **Technology used**

* React.js (NPM Package)
* Tailwind CSS (NPM Package)
* Daisy UI (NPM Package)
* JavaScript (ES6+)
* Vite (NPM Package)
* Lucide icons (NPM Package)
* React Toastify (NPM Package)
* Google Fonts
* JSON (For product data)
* Github
* Netlify (Project Deployment)

###### **Key Features in this Projects**

1. **Smart Cart System**

--- Users can add products to the cart and instantly see updates without refreshing the page. The cart dynamically calculates total price and quantity.

2. **Tab-Based Navigation** 

--- The interface includes a smooth tab system (Products & Cart) that allows users to switch views easily without reloading the page.

3. **Clean & Responsive UI**

--- Built with Tailwind CSS, the design is fully responsive and looks great on desktop, tablet, and mobile devices
